# 飞机大战 #
## 基本 ##
作者QMMMS  
游戏主程序是plane_main  
要玩就打开它  
默认帧率60  
你可能会发现Python解释器会输出一些 warning  
不用在意  
它与bmp图片有关  
## 操作 ##
游戏基本操作是键盘，上下左右键  
如果你与敌机相撞  
你将死亡  
程序将自动结束  
## 提示 ##
工具文件plane sprites  
qms test v1、2、3是我自己测试用的  
与游戏主体无关  
我没有写注释
